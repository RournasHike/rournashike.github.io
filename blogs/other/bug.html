    # 使用vuepress主题theme-cli

    # 使用theme-cli解决Error: unable to verify the first certificate报错

    ① 取消ssl验证：npm config set strict-ssl false -g或者yarn config set strict-ssl false -g

    如果还没成功，则将npm源更换为国内镜像：

    npm config set registry http://registry.npm.taobao.org/ -g

npm config set registry http://registry.cnpmjs.org/ -g

    网络一定要好否则使用theme-cli脚手架初始化项目失败

    # npm run dev运行项目失败xxx is not a valid name

    原因是建立项目的project name写错了

    解决：

    1.在项目package.json文件夹下修改name属性，尽量不要有空格，英文单词即可

    2.清除项目建立后，npm install的node_modules，重新安装

    # npm 删除node_modules

    • 点击鼠标右键打开Git Bash Here， 输入指令 npm install -g remove-node-modules

    • 然后找到要删除的项目根目录下右键点击Git Bash Here（也可：cd 拉取项目文件夹-回车），然后输入：remove-node-modules即可删除。

    # 重新安装依赖
    npm install
